## 常用的命令行指令
### 一.打开cmd的方式
1.win+R键

2.在任意文件夹下面，按住shift键+鼠标右键点击，在此处打开命令行窗口

3.在资源管理器的地址栏上面加上cmd+空格后面跟上路径

4.windows系统里面命令提示符右键以管理员身份运行

### 二.常用DOS命令
1.切换盘符
```
D:
```
2.查看目录
```
dir
```
3.切换目录

跨盘符切换使用参数/d
```
cd /d D:\coding
```

同盘符切换

（返回上一级目录）
```
cd ..
```
（切换到子目录）
```
cd  coding\coder
```
4.清理屏幕
```
cls
```
5.退出终端
```
exit
```
6.查看电脑ip
```
ipconfig
```
7.ping指令
```
ping 'url'
```
8.操作文件夹及文件
新建文件夹：
```
md '文件夹名'
```
新建文件：
```
cd>'文件名'
```
删除文件
```
del '文件名'
```
删除文件夹
```
rd '文件夹名'
```
